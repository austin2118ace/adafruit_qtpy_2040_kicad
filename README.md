# Adafruit QT Py 2040 KiCad Symbol and Footprint

This repository contains a symbol and footprint for the Adafruit QT Py 2040 module. This was a conversion from the original boardfiles and schematic that can be found here: https://github.com/adafruit/Adafruit-QT-Py-RP2040-PCB

3D Model for the footprint was taken from here: 
https://github.com/adafruit/Adafruit_CAD_Parts/tree/main/4900%20QTPy%20RP2040

## Installation
- Add the symbol to your symbol libraries
- Add the footprint to your footprint libraries
